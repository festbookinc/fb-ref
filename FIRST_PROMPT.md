페스트북 아카이브 사이트
26.03.01

나는 비개발자고 Cursor AI를 활용해 Next.js와 Tailwind CSS로 이미지 아카이브 사이트를 만들 거야. 기술 스택은 Next.js, Supabase(DB/Storage/Auth), Vercel 배포를 기준으로 단계별로 코드를 짜줘.

자세한 내용은 다음과 같아.
우선 작업 순서를 기획하고, 나에게 공유를 한 다음, 하나씩 차례대로 진행하자

목적
구성원들이 이미지 레퍼런스를 자유롭게 업로드하고 공유
레퍼런스 자료들을 검색 및 태그 기능으로 자유롭게 서치
따로 보드를 만들어서 관리할 수 있음 (일종의 폴더처럼 사용)

제작 방식
비개발자가 cursor ai 를 활용하여 바이브코딩
필수 기능부터 순차적으로 개발. AI의 개발과 사용자의 테스트를 번갈아 진행.
festbookinc git 에 올린 후 vercel로 deploy 예정
이미지 업로드와 데이터 저장을 위해 Supabase를 사용
Lazy Loading: 화면에 보이는 이미지만 먼저 로드
WebP 변환: 업로드 시 자동으로 용량이 적은 WebP 포맷으로 변환.

UI 테마
블랙 앤 화이트의 모던 갈끔
완전 화이트가 아닌 옅은 회색 배경에 진한 회색 글씨
다크 모드 기능 지원
세련, 정제되고 섬세한 모션 인터랙션

UI 구조
네비게이션 바와 이미지 아카이브 영역, 2가지로 구성

네비게이션 바:
[로고] [검색창] [태그 버튼] [추가 버튼] [로그인 버튼 or 마이페이지 버튼]

본문 이미지 아카이브 영역:
이미지 카드들이 좌우로 꽉차게 배열 (max width 없음. 모바일에선 2단 그리드, PC에서는 화면 창에 맞춰서 4단 5단 6단 등 계속 늘어나게)
이미지 카드 크기 조절 가능
우상단에 정렬 방식 드롭다운 버튼과 이미지 카드 크기 조절바

개별 이미지 카드 영역:
이미지 미리보기는 5:7 비율, 클릭할 경우 원본 크기 모달 창으로
텍스트 박스: 제목, 설명, 태그, 작성자
다운로드 버튼, 편집 버튼(작성자만 사용 가능), 보드에 추가 버튼(추가하면서 생성도 가능)
댓글 칸
댓글 작성 창

기능
로고
클릭하면 메인 페이지로 이동
로고는 임시 텍스트 사용 ‘FB Ref.’

검색창
제목, 설명, 태그, 작성자, 댓글 내용 등을 검색할 수 있음

태그 버튼
클릭하면 태그 선택 모달창
태그를 선택하여 해당 태그에 해당하는 이미지만 모아서 볼 수 있음
복수 선택 가능
태그명 검색 가능

추가 버튼
클릭하면 추가 정보 입력 모달창
입력값:
[이미지(파일 업로드 or 이미지 링크) - 필수, 
제목 - 필수, 
설면, 
링크, 
태그(여러개 가능)]
파일 한 번에 올리기 기능 있음 - 여러개의 파일을 올림. 파일명이 자동으로 제목 선입력하되 수정 가능, 나머지는 별도로 입력할 수 있도록
태그의 경우, 사용자가 입력한 텍스트 값이 하나의 태그가 됨. 자동 완성 기능도 추가 (‘그래픽’ 이라는 태그가 이미 있는 경우, ‘ㄱ’ 혹은 ‘그’를 입력했을 때 ‘그래픽’이라는 완성 텍스트가 드롭 박스 형태로 뜨도록)

로그인 버튼
클릭하면 모달창 - 로그인창과 회원가입 버튼
festbook google 계정 사용자만 회원가입 할 수 있도록
NextAuth.js를 이용한 Google Login 추천. 특정 도메인(@festbook.inc 등)만 가입 가능하도록 제한하는 로직을 추가
로그인 유지 기능
로그인 후에는 해당 위치에 마이페이지 버튼으로

마이페이지 버튼
로그인 후 로그인 버튼이 있던 자리에 배치
클릭하면 마이페이지로 이동
본인이 업로드한 자료들 모아보는 곳
이미지를 따로 모아보는 보드 생성 가능

이미지 아카이브 영역
좌상단에 정렬 방식 드롭다운 버튼
정렬 방식 종류: 최신순, 랜덤순
인피니트 스크롤 (Infinite Scroll)

이미지 카드 크기 조절바
최소 100px-최대 800px 로 조절 가능

이미지 카드
이미지 미리보기는 5:7 비율, 클릭할 경우 원본 크기 모달 창으로
텍스트 박스: 제목, 설명, 태그, 작성자
다운로드 버튼
편집 버튼(작성자만 사용 가능): 내용 편집 및 삭제 기능
보드에 추가 버튼(추가하면서 생성도 가능)

드래그 앤 드롭 업로드 기능
메인 화면 아무 곳에나 이미지를 끌어다 놓으면 업로드 모달이 뜸
